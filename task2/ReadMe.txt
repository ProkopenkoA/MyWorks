Программа считает парматеры заданной фигуры по ее старонам.
Как параметры в консоле задается файл откуда будет читаться Имя фигуры и её отрезки "-i <nameFile>" .
Так же задается параметр куда выводиться информация о фигуре в консоле "-c" или в файл "-o <nameFile>"
Файл для чтение должен содержать в первой строке Имя фигуры на английском языке большими буквами,
во втрой строке парматеры фигуры через пробел
Пример ввода в консоль:
-i input.txt -o output.txt ...
Пример файла:
Circle
5
